### Variable Dictionary

* **hotel** — Hotel type (H1 = Resort Hotel, H2 = City Hotel)
* **is_canceled** — Booking canceled (1) or not (0)
* **lead_time** — Days between booking and arrival

**Arrival Date**

* **arrival_date_year** — Arrival year
* **arrival_date_month** — Arrival month
* **arrival_date_week_number** — Arrival week number
* **arrival_date_day_of_month** — Arrival day

**Stay**

* **stays_in_weekend_nights** — Weekend nights booked
* **stays_in_week_nights** — Weekday nights booked

**Guests**

* **adults** — Number of adults
* **children** — Number of children
* **babies** — Number of babies

**Booking Details**

* **meal** — Meal package type
* **country** — Country of origin
* **market_segment** — Market segment
* **distribution_channel** — Booking channel
* **is_repeated_guest** — Repeat guest (1/0)

**Customer History**

* **previous_cancellations** — Past cancellations
* **previous_bookings_not_canceled** — Past non-canceled bookings

**Room**

* **reserved_room_type** — Reserved room code
* **assigned_room_type** — Assigned room code

**Management**

* **booking_changes** — Number of booking changes
* **deposit_type** — Deposit category
* **agent** — Travel agency ID
* **company** — Company ID
* **days_in_waiting_list** — Days in waiting list

**Customer Type**

* **customer_type** — Booking type

**Pricing**

* **adr** — Average Daily Rate

**Requests**

* **required_car_parking_spaces** — Parking spaces needed
* **total_of_special_requests** — Special requests count

**Status**

* **reservation_status** — Final status
* **reservation_status_date** — Status date
